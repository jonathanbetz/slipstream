# Slipstream

## Tool preferences

- Prefer the **Grep** tool over `Bash(grep ...)` for searching files and code.
- Prefer the **Glob** tool over `Bash(ls ...)` for listing and exploring directories.

## Development workflow

- To deploy hooks and commands after editing, run `./install.sh` — do not manually
  `cp` individual files to `~/.claude/hooks/` or `~/.claude/commands/`.
